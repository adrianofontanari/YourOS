# 👤 About Me

> [!tip] This data view query lists all the key pages connected with the diary (with the tag 'aboutme'. It serves as a central repository to document your journey and life-learnings.

```dataview
TABLE WITHOUT ID (link(file.path)) as Diary
FROM #aboutme
```
